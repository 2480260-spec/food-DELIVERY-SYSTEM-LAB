# Functional Requirements (FR)
* FR-1: User registration aur login functionality.
* FR-2: Menu browse karne aur search karne ki sahulat.
* FR-3: Cart mein items add karna aur checkout karna.
* FR-4: Real-time order tracking status.
* FR-5: Multiple payment gateways (Credit Card, JazzCash, etc.) ka support.
# Non-Functional Requirements (NFR)
* NFR-1: System response time 2 seconds se kam hona chahiye.
* NFR-2: System 5000 concurrent users ko handle kar sake.
* NFR-3: User ka data fully encrypted hona chahiye (Security).
* NFR-4: System 99.9% uptime provide kare (Availability).
* NFR-5: Mobile aur Desktop dono screens par UI sahi nazar aaye (Portability).
